# Sysproxy

## Usage
```
% ./sysproxy -help
Usage of ./sysproxy:
  -bypass string
        bypass join by ,
  -http string
        http proxy server and port
  -https string
        https proxy server and port
  -setDNS string
        dns servers join by ,
  -show
        show all proxies
  -showDNS
        show dns servers
  -socks string
        socks proxy server and port
  -stop
        disable all proxies
```
